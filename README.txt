INSTALLATION INSTRUCTIONS:
═══════════════════════════════════════════════════════════

1. Install Python 3.8 or higher from https://python.org
   ☑️ Check "Add Python to PATH" during installation

2. Open Command Prompt or PowerShell in this folder

3. Install required library:
   pip install -r requirements.txt

4. Double-click run_sysinfo.bat

═══════════════════════════════════════════════════════════

USAGE:
═══════════════════════════════════════════════════════════

[1] Show System Information Report - View CPU, RAM, Disk info
[2] Organize Files in a Folder - Auto-sort files by type
[3] Exit Application

═══════════════════════════════════════════════════════════

REQUIREMENTS:
═══════════════════════════════════════════════════════════

• Windows 10/11
• Python 3.8 or higher
• psutil library (installed automatically in step 3)
